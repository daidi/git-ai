## ⚡️ Nỗi Đau Lớn Chờ Đợi

Dùng các plugin hay tính năng AI khác, bạn buộc phải nhấn tạo, rồi ngồi nhìn tiến độ xoay như chong chóng. Khựng lại không code tiếp được, giảm năng suất cực đoan.

## 🚀 Git AI - Thay Khớp Phá Vỡ

Cấu trúc chạy ngầm (Asynchronous) của Git AI tách rời UI với AI.

Bạn gõ trên terminal 2 từ hời hợt:
`git commit -m "fix loi"`

Và bạn code tiếp dự án! Trong lúc đó Git AI ngầm chuyển file đi, viết câu lệnh Conventional thần sầu và âm thầm thay thế vào Git:
`fix(auth): handle null exception in token response`

## 📦 Có cả tiện ích mở rộng IDE

Sử dụng trực tiếp trong VS Code bằng cách tìm kiếm Git AI trên Marketplace, hoặc trên Jetbrains IntelliJ IDEA. Cài đặt khoá API dễ dàng bằng đồ hoạ. Cài là ngấm luôn.
